# Cross-Protocol Loan UI Guide

1. Navigate to the Borrow section.
2. Choose your collateral source (e.g., Morpho Earn Vault) and target borrow asset (e.g., USDC on Aave).
3. If needed, zap into a Morpho position directly from the flow.
4. Review the estimated health factor and borrow rates.
5. Confirm to establish the position.
6. Monitor and manage LTV and health factor over time.

> Source: [Cross-Protocol Loan UI guide](https://246-club.gitbook.io/docs)
